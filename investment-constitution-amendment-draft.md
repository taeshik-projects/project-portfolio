## 1️⃣ Option 1: DCA 극한 가속 규칙

### 현행 (투자 헌법 v2.2)
```
- 정상 시장: 10주 DCA
- 급락 시 (S&P -15%): 5주로 단축
- 과열 시 (CAPE > 35): 15주로 연장
```

### 개정안 (v3.0)
```
Level 1 - 정상: 10주 DCA
Level 2 - 약한 공포 (S&P -10~15%): 7주
Level 3 - 중간 공포 (S&P -15~25%): 5주
Level 4 - 강한 공포 (S&P -25~35%): 3주
Level 5 - 극한 공포 (S&P -35% 이상): 2주 ⚡️
```

**추가 트리거 (VIX 기준):**
- VIX 30-40: Level 3
- VIX 40-60: Level 4
- VIX 60+: Level 5 (2008, 2020급)

**실행 예시:**
```
시나리오: 나스닥 -30% 급락 (2020년 3월급)
- 현행: 5주 DCA ($10k → 주당 $2k)
- 개정: 3주 DCA ($10k → 주당 $3.3k)
- 효과: 2주 빠르게 저점 포착
```

**안전 장치:**
- 현금 버퍼 최소 **$5,000** 사수 (현행 $10k → 완화)
- 급락 중에도 절대 현금 $5k 아래로 안 감
- Level 5 발동 시 현금 버퍼 $3k까지 허용 (극한 상황만)

## 2️⃣ Option 4: 제한적 레버리지 예외 조항 ⚠️

### 원칙
> "레버리지는 독약이다. 하지만 2008/2020급 공포는 한 세대에 2-3번뿐이다."

### 발동 조건 (모두 충족 필수)
```
✅ 1. S&P 500 고점 대비 -30% 이상 급락
✅ 2. VIX > 50 (극단 공포)
✅ 3. 침체 체크리스트 5개 중 4개 이상
✅ 4. 대식님 본인이 "이건 진짜 기회다" 판단 (최종 승인)
```

**예시:**
- 2008년 금융위기 (-57%)
- 2020년 코로나 (-34%)
- 2022년 베어마켓 (-25%, ❌ 조건 미달)

### 레버리지 유형 및 상한

#### A) Non-Mortgage Loan (신용 점수 + 투자 겸용) ⭐️ **추천**
```
목적:
- FICO Score 부스트 (installment loan 기록)
- 투자 레버리지 (저금리 시 기회 포착)

조건:
- 대출 규모: 총자산의 10% 이하
- 최대 금액: $30,000 (현 자산 기준 ~$27k)
- 이자율: 6% 이하 (신용 점수 좋으면 4-5% 가능)
- 상환 기간: 12-24개월 (장기 신용 기록)
- 월 상환액: 급여의 10% 이하 (부담 없는 수준)

투자 전략:
- 대출금 100%를 즉시 투자 (3주 DCA)
- 월 상환금은 급여에서 자동 납부
- 시장 회복 시 차익 실현 → 조기 상환 가능

예시 (자산 $273k 기준):
- 대출: $25,000 @ 5% APR (24개월)
- 월 상환: ~$1,100 (급여 대비 충분히 안전)
- 투자: QQQ/VOO 혼합 (3주 DCA)
- 기대 수익: 연 10% (보수적) → 5년 후 +$12,500
- 대출 이자: -$1,300 → 순이익 +$11,200
- 신용 점수: +20-40점 (installment loan 기록)
```

**장점:**
- ✅ 신용 점수 향상 (본래 목적)
- ✅ 투자 기회 포착 (부수 효과)
- ✅ 마진콜 없음 (일반 대출)
- ✅ 고정 이자율 (리스크 예측 가능)

**단점:**
- ⚠️ 고정 상환 의무 (시장 더 떨어져도 갚아야 함)
- ⚠️ 조기 상환 수수료 가능 (확인 필요)

#### B) Securities-Based Loan (증권 담보 대출)
```
조건:
- 대출 규모: 포트폴리오의 20% 이하 (은행 일반적 한도 50%)
- 이자율: 4-7% (Prime Rate 기준)
- 마진콜 트리거: LTV 35% (보수적)

예시:
- 포트폴리오: $200k
- 대출: $40k (20%)
- 마진콜: $200k → $114k (-43% 추가 하락 시)
```

**장점:**
- ✅ 저금리 (4-5%)
- ✅ 유연한 상환 (부담 적음)

**단점:**
- ❌ 마진콜 리스크 (강제청산 가능)
- ❌ 신용 점수 무관 (installment loan 아님)
- ❌ 심리적 압박 (하락 시 공포 증폭)

#### C) 레버리지 ETF ❌ **여전히 금지**
```
이유:
- 일일 리밸런싱 → 장기 보유 시 성과 왜곡
- 극단 변동성 (-90% 가능)
- 투자가 아니라 투기
```

### 추천 우선순위
```
1순위: Non-Mortgage Personal Loan (신용+투자 겸용) ⭐️
2순위: Securities-Based Loan (투자만, 신중히)
3순위: 레버리지 ETF (절대 금지)
```

### 실행 체크리스트 (발동 시)

**Phase 1: 조건 확인 (필수)**
```
[ ] S&P -30% 이상?
[ ] VIX > 50?
[ ] 침체 체크리스트 4/5?
[ ] 대식님 최종 승인?
[ ] 현금 버퍼 $3k 이상 남음?
```

**Phase 2: 대출 실행**
```
[ ] 은행 대출 조건 비교 (BoA, Fidelity, SoFi 등)
[ ] 이자율 6% 이하 확인
[ ] 월 상환액 급여 10% 이하 확인
[ ] 대출 계약서 최종 검토
[ ] 승인 후 3일 이내 투자 실행
```

**Phase 3: 투자 집행**
```
[ ] 3주 DCA로 분할 매수 (감정 차단)
[ ] 70% 코어 (VOO/QQQ)
[ ] 20% 테크 (SMH)
[ ] 10% 해외 (VEA/VWO)
[ ] 매수 후 기록 (금액/날짜/가격)
```

**Phase 4: 모니터링 (월 1회)**
```
[ ] 포트폴리오 수익률 확인
[ ] 대출 상환 정상 진행 중?
[ ] 조기 상환 가능 시점 도래?
[ ] 신용 점수 변화 추적
```

**Phase 5: 출구 전략**
```
조기 상환 트리거:
- 포트폴리오 +20% 수익 → 일부 매도 후 상환
- 보너스 수령 시 → 즉시 상환
- 시장 재과열 (CAPE > 35) → 상환 후 비중 축소

최종 상환:
- 24개월 만기 도래 시 자동 상환
- 신용 기록 유지 (1년 이상 보유 필수)
```

### 절대 금지 사항 🚨
```
❌ 대출금으로 비트코인/개별주 매수
❌ 조건 불충족 시 "감으로" 발동
❌ 대출 연장/추가 차입 (한 번만 허용)
❌ 레버리지 ETF 매수
❌ 현금 버퍼 $3k 아래로 하락
```

## 3️⃣ 신용 점수 최적화 전략 (Non-Mortgage Loan)

### 목표
- **단기 (6개월):** FICO 780 → 820+
- **장기 (2년):** 850 도전

### 전략 1: Personal Loan 활용 ⭐️

**Normal 시장 (공포 아닐 때):**
```
목적: 순수 신용 점수 향상 (투자 X)

방법:
- 소액 Personal Loan ($5,000-10,000)
- 이자율 무관 (5-8% OK)
- 즉시 HYSA에 보관 (5% 수익)
- 24개월 자동 상환 설정
- 만기까지 보유 (중도 상환 X)

효과:
- Installment loan 기록 생성
- Credit mix 개선
- 평균 계좌 연령 증가
- 순비용: 연 $100-300 (이자 차액)
```

**Extreme 공포 (조건 충족 시):**
```
목적: 신용 점수 + 투자 레버리지 겸용

방법:
- 대규모 Personal Loan ($20,000-30,000)
- 저금리 필수 (4-6%)
- 100% 투자 집행 (3주 DCA)
- 수익 발생 시 조기 상환 고려

효과:
- 신용 점수 +20-40점
- 투자 수익 기대
- 순비용/이익: 시장 성과에 따라
```

### 전략 2: 신용카드 최적화 (병행)
```
현재 문제:
- Revolving credit만 있음 (카드 결제 이력)
- Installment credit 없음 (할부 대출 이력)

해결책:
- Personal Loan으로 installment 추가 ✅
- 신용카드 사용률 10% 이하 유지 (현재 OK)
- 오래된 카드 절대 해지 X
```

### 실행 플랜 (시나리오별)

#### 시나리오 A: 정상 시장 (지금)
```
Step 1: 소액 Personal Loan 신청
- 금액: $5,000-10,000
- 은행: BoA (기존 고객), Marcus, SoFi

Step 2: 승인 후
- HYSA에 보관 (Capital One 360, 5% APY)

Step 3: 모니터링
- 3개월 후 FICO 확인
- 목표: 800+ 달성

#### 시나리오 B: 극한 공포 발동 (조건 충족 시)
```
Step 1: 대규모 Personal Loan 신청
- 금액: $25,000-30,000
- 저금리 협상 (신용 점수 780+)
- 목적: 신용 + 투자 겸용

Step 2: 승인 후 즉시 투자
- 3주 DCA 집행

Step 3: 수익 관리
- 6개월 후 포트폴리오 평가

Step 4: 신용 점수 재확인
