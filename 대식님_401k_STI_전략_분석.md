## 1. STI 보너스 세금 처리 방식

### 1.1 보너스의 세법상 분류
**IRS 규정에 따른 처리**:
- 보너스는 **Supplemental Wage (부가급여)**로 분류됩니다
- 일반 급여(Regular Wage)와는 **별도의 원천징수 방식** 적용
- W-2 Box 1에 일반 급여와 합산되어 보고됨

**소스**: IRS Publication 15 (Circular E), 2026

### 1.2 Supplemental Wage Withholding 방식

IRS는 고용주에게 두 가지 원천징수 방법을 제공합니다:

#### 방법 1: Percentage Method (고정 세율 방식)
**가장 일반적으로 사용되는 방법**

```
보너스 금액에 따른 원천징수율:
• $1,000,000 이하: 22% 고정세율
• $1,000,000 초과분: 37% 고정세율
```

**예시 계산**:
- 대식님 보너스: $50,000
- 연방세 원천징수: $50,000 × 22% = **$11,000**
- 실제 수령액: $50,000 - $11,000 - (FICA 등) = **약 $37,350**

#### 방법 2: Aggregate Method (합산 방식)
- 보너스를 정규 급여와 합산하여 세금 계산
- W-4 정보 기반으로 누진세율 적용
- 일시적으로 더 높은 세율 구간으로 이동 가능

### 1.3 일반 급여와의 차이점

| 구분 | 일반 급여 (Regular Wage) | 보너스 (Supplemental Wage) |
|------|-------------------------|---------------------------|
| 원천징수 방식 | W-4 기반 누진세율 | 22% 고정 또는 합산 방식 |
| 세율 결정 | 개인 W-4 정보 | 고정세율 (일반적) |
| 최종 세금 | 연말정산 시 조정 | 연말정산 시 조정 |
| 401k 기여 | 가능 | 가능 |

**중요**: 원천징수된 22%는 **예상 세금**이며, 실제 세금은 연말정산 시 총 소득에 따라 결정됩니다.

## 2. STI → 401k 전략 비교

### 2.1 시나리오 설정
**가정**:
- 대식님 연봉: $180,000
- STI 보너스: $50,000
- 기혼 합산 신고 (Married Filing Jointly)
- 2026년 세율: 24% 구간 ($201,050 - $383,900)

### 2.2 시나리오 A: STI → 401k Roth

#### 세금 처리 과정
```
1단계: 보너스 수령
  - 보너스 총액: $50,000
  - 연방세 원천징수 (22%): -$11,000
  - FICA (7.65%): -$3,825
  - 실제 수령액: $35,175

2단계: Roth 401k 기여
  - 기여 가능액: $23,500 (2026년 한도)
  - 추가 필요 금액: $23,500 - $35,175 = 불가능
  ※ 실제 수령액으로는 401k 한도를 채우기 어려움
```

#### 장기 효과 (30년 후 은퇴 시)
```
가정: 7% 연평균 수익률, 30년 투자

초기 투자: $23,500 (Roth)
미래 가치: $178,873
은퇴 시 인출: $178,873 (세금 $0) ← 핵심 장점
실질 수령: $178,873
```

**장점**:
- ✅ 은퇴 시 인출 금액 전액 세금 면제
- ✅ 미래 세율 상승 리스크 제거
- ✅ 상속 시 수혜자에게 유리 (세금 없는 자산)

**단점**:
- ❌ 현재 높은 세율(24%)로 세금 납부
- ❌ 보너스 원천징수 후 실제 수령액 부족
- ❌ 즉각적인 세금 절감 효과 없음

### 2.3 시나리오 B: STI → 401k Pre-tax

#### 세금 처리 과정
```
1단계: Pre-tax 401k 기여 (보너스 전액)
  - 보너스 총액: $50,000
  - 401k 기여액: $23,500 (2026년 한도)
  - 과세 대상 보너스: $26,500
  
2단계: 실제 수령 계산
  - 과세 보너스: $26,500
  - 연방세 원천징수 (22%): -$5,830
  - FICA (7.65%): -$3,825 (전체 보너스 대상)
  - 실제 수령액: $16,845
  
3단계: 세금 절감 효과
  - 소득 감소: $23,500
  - 절감 세금 (24% 구간): $23,500 × 24% = $5,640
  - 즉시 현금 흐름 개선: +$5,640
```

#### 장기 효과 (30년 후 은퇴 시)
```
초기 투자: $23,500 (Pre-tax)
미래 가치: $178,873
은퇴 시 세율 가정: 15% (은퇴 후 소득 감소)
세금: -$26,831
실질 수령: $152,042

차이: $178,873 - $152,042 = $26,831 더 적음
```

**장점**:
- ✅ 즉각적인 세금 절감 ($5,640)
- ✅ 현재 높은 세율에서 공제
- ✅ 보너스 전액 활용 가능
- ✅ 현재 현금 흐름 개선

**단점**:
- ❌ 은퇴 시 인출 시 세금 부과
- ❌ 미래 세율 상승 리스크
- ❌ 총 수령액 잠재적 감소

### 2.4 수치 비교 요약

| 구분 | Roth 401k | Pre-tax 401k | 차이 |
|------|-----------|--------------|------|
| 현재 세금 절감 | $0 | **$5,640** | +$5,640 (Pre-tax) |
| 보너스 실수령 | $35,175 | $40,670 | +$5,495 (Pre-tax) |
| 30년 후 총액 | $178,873 | $178,873 | 동일 |
| 은퇴 시 세금 | $0 | $26,831 | +$26,831 (Roth) |
| **최종 수령액** | **$178,873** | **$152,042** | **+$26,831 (Roth)** |

**결론**: 
- **단기적으로는 Pre-tax가 유리** (현금 흐름)
- **장기적으로는 Roth가 유리** (총 수령액 $26,831 더 많음)
- 하지만 대식님 같은 고소득자는 **Pre-tax가 일반적으로 권장됨**

## 3. 401k Max Out 전략 비교

### 3.1 전략 1: STI로 401k 채우기

**실행 방법**:
```
시나리오:
- 정규 급여 기여: 연간 $0
- STI 보너스 기여: $23,500 (401k 한도 전액)
- 보너스 수령 시기: 연말 (12월)
```

#### 장점
✅ **즉각적 세금 절감**
   - 보너스 $23,500 공제
   - 절감액: $5,640 (24% 세율 가정)

✅ **보너스 높은 세율에 대응**
   - Supplemental wage 22% 원천징수 회피
   - 고정세율 부담 감소

✅ **연말 일시 투자**
   - 시장 타이밍 단순화
   - 관리 편의성

#### 단점
❌ **Dollar Cost Averaging 효과 상실**
   - 연중 가격 변동 활용 불가
   - 시장 하락기 매수 기회 상실

❌ **Employer Match 손실 가능**
   - 많은 회사가 "매 급여당 매칭" 정책
   - 연중 기여 없으면 매칭 손실
   - **예시**: 6% 매칭 시 연간 $10,800 손실 가능

❌ **현금 흐름 불균형**
   - 연중 401k 없어서 실수령액 높음
   - 연말 보너스 시 실수령액 급감

❌ **유동성 리스크**
   - 연중 예상치 못한 지출 발생 시 대응 어려움

### 3.2 전략 2: 샐러리로 401k Max Out

**실행 방법**:
```
시나리오:
- 연봉: $180,000 (월 $15,000)
- 월 401k 기여: $1,958 ($23,500 ÷ 12개월)
- STI 보너스: 일반 소득으로 수령
```

#### 장점
✅ **Dollar Cost Averaging 효과**
   - 매월 일정액 투자
   - 시장 변동성 완화
   - 장기 수익률 안정화

✅ **Employer Match 극대화**
   ```
   예시: 6% 매칭 시
   - 월 급여: $15,000
   - 월 매칭: $15,000 × 6% = $900
   - 연간 매칭: $10,800 (무료 돈!)
   ```

✅ **현금 흐름 안정화**
   - 매월 일정한 급여 패턴
   - 예산 관리 용이
   - 보너스는 별도 용도 활용

✅ **보너스 유연성 확보**
   ```
   보너스 $50,000 활용 가능 옵션:
   1. 비상 자금 (3-6개월 생활비)
   2. 주택 다운페이먼트
   3. 부채 상환 (고금리 우선)
   4. Taxable 계좌 투자
   5. 자녀 교육 자금 (529 plan)
   ```

#### 단점
❌ **보너스 세금 부담**
   - 22% 원천징수 적용
   - 즉각적 세금 절감 없음

❌ **월별 실수령액 감소**
   - 401k 기여로 급여 감소
   - 생활비 압박 가능

### 3.3 하이브리드 전략 (권장)

**최적 조합**:
```
1단계: 정규 급여로 Employer Match 확보
   - 월 기여: $900 (급여의 6%)
   - 연간 기여: $10,800
   - 연간 매칭: $10,800 (100% 수익!)

2단계: STI로 401k 한도 채우기
   - 보너스 중 401k 기여: $12,700
   - 총 401k 기여: $23,500 (한도 달성)
   - 세금 절감: $3,048

3단계: 남은 보너스 활용
   - 남은 보너스: $37,300
   - 세후 수령액: 약 $28,000
   - 용도: 비상자금/투자/목표저축
```

**장점 종합**:
- ✅ Employer match 전액 확보 ($10,800)
- ✅ 401k 한도 달성 ($23,500)
- ✅ 세금 최적화
- ✅ 유동성 확보 ($28,000)
- ✅ Dollar cost averaging 효과

## 4. 대식님께 최적 전략 추천

### 4.1 대식님 상황 분석

**프로필**:
- 연령: 36세 (은퇴까지 약 29년)
- 직업: ASML 엔지니어 (고소득 안정직)
- 예상 소득: $180,000+ (상위 15% 이내)
- 세율 구간: 24% (Married Filing Jointly)
- STI 보너스: 연간 지급

**세율 분석**:
```
현재 세율: 24% (연소득 $230,000 가정)
은퇴 후 예상 세율: 12-15%

근거:
1. 은퇴 후 소득원
   - Social Security: $40,000-50,000
   - 401k 인출: $60,000-80,000
   - 총 소득: $100,000-130,000
   - 예상 세율: 12-15% 구간

2. 세율 차이
   - 현재 vs 은퇴: 24% vs 12-15%
   - 차이: 9-12%포인트
   → Pre-tax가 유리!
```

```
   - Triple tax advantage
```

### 4.3 Roth가 유리한 예외 상황

**다음의 경우 일부 Roth 고려**:
```
1. 미래 세율 상승 예상
   - 소득 급증 예상 (승진, 이직)
   - 배우자 소득 증가
   - 세법 변경 (세율 인상)

2. 상속 계획
   - 자녀에게 세금 없는 자산 상속
   - Roth IRA는 상속세 유리

3. 은퇴 소득이 높을 것으로 예상
   - 부동산 임대 소득
   - 부업/컨설팅 소득
   - 연금 소득 과다

권장 비율: Pre-tax 80% + Roth 20%
```

```
│ 실질 차이: $496,073 - $427,892          │
│          = $68,181 (Roth 소폭 유리)    │
└─────────────────────────────────────────┘
```

**결론**: 
- 장기적으로 Roth가 약간 유리할 수 있으나
- 불확실성과 현재 현금 흐름 고려 시
- **Pre-tax가 실용적 선택**
