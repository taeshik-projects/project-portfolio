# 🎻 악보 프로젝트 마일스톤 (편곡 중심)

**프로젝트:** 대중음악 편곡 악보 제작 및 판매
**목표:** 부업으로 시작 → 월 $1,000+ 수익 (12개월 내)
**핵심 전략:** 편곡 90% + 채보 10%

---

## Phase 1: 검증 및 준비 (0-2개월)

### M1. 시장 검증 및 타겟 선정 ✅
**목표:** 수익성 있는 틈새 시장 파악
- [x] 대중음악 string quartet/violin duo 편곡 수요 확인
- [x] 타겟 장르 결정 (K-pop, 팝, 웨딩송, 애니OST)
- [x] 경쟁사 가격 조사 완료
**상태:** 완료 (2026-02-13 분석 완료)

### M2. 편곡 워크플로우 확립 [편곡]
**목표:** 효율적인 제작 시스템 구축
- [ ] 기존 악보 소스 확보 방법 정립
  - Musicnotes, SheetMusicDirect 계정 생성
  - IMSLP (퍼블릭 도메인) 리서치
  - 한국 악보 사이트 조사
- [ ] 편곡 툴 선정 및 마스터
  - MuseScore (무료) vs Dorico vs Finale 비교
  - 최종 선택 및 워크플로우 세팅
- [ ] 테스트 편곡 1개 완성
  - 곡 선정: 간단한 팝송 (예: "Can't Help Falling in Love")
  - 피아노 악보 → String Quartet 4파트
  - 연주 가능성 검증 (본인 연주 또는 오케스트라 동료)
- [ ] 편곡 시간 측정
  - 곡당 실제 소요 시간 기록 (목표: 3-5시간/곡)
**완료 조건:** 테스트 편곡 1개 + 시간 데이터

### M3. 품질 기준 및 가격 정책 [편곡]
**목표:** 일관된 품질 및 수익성 확보
- [ ] 연주 가능성 체크리스트 작성
  - 음역대 확인 (바이올린 G3-E7)
  - 핑거링/포지션 이동 고려
  - 보잉 기술 (레가토, 스타카토 등) 표기
  - 파트 간 밸런스 확인
- [ ] 난이도별 가격표 수립
  - 초급 (2-3분): $15
  - 중급 (3-4분): $20-25
  - 고급 (4분+, 복잡): $30+
- [ ] 샘플 파일 포맷 정의
  - Full Score PDF + 개별 파트 PDF
  - 워터마크 샘플 (처음 30초)
  - 미리듣기 MP3 (MIDI 렌더링)
**완료 조건:** 체크리스트 문서 + 가격표 완성

## Phase 2: MVP 및 초기 카탈로그 (2-4개월)

### M4. 첫 10개 편곡 악보 제작 [편곡]
**목표:** 판매 가능한 초기 카탈로그 구축
- [ ] **String Quartet (5곡)**
  - 웨딩송 2곡: "Perfect" (Ed Sheeran), "Spring Day" (BTS)
  - 인기 팝 2곡: "A Thousand Years", "Shallow"
  - 애니/OST 1곡: "Married Life" (Up)
- [ ] **Violin Duo (3곡)**
  - 학생용 중급: "Let It Go" (Frozen), "City of Stars"
  - 고급 1곡: "Bohemian Rhapsody" (일부)
- [ ] **기타 편성 실험 (2곡)**
  - Violin + Cello: 1곡
  - Violin + Piano: 1곡
- [ ] 각 곡 품질 검증
  - 체크리스트 적용
  - 동료 연주자 피드백 1명 이상
**완료 조건:** 10개 악보 PDF + MP3 완성

### M5. 온라인 판매 시작 [편곡]
**목표:** 첫 매출 발생 및 시장 반응 확인
- [ ] 판매 플랫폼 선정 및 입점
  - Option A: Musicnotes/SheetMusicDirect (심사 필요)
  - Option B: Gumroad/Etsy (즉시 시작 가능)
  - Option C: 개인 사이트 (Shopify/WordPress)
- [ ] 개인 포트폴리오 사이트 오픈
  - 도메인 구매 (예: taeshik-arrangements.com)
  - 악보 샘플 갤러리
  - 구매 링크 연결
- [ ] 마케팅 초기 실행
  - Reddit (r/violinist, r/ClassicalMusic) 포스팅
  - Facebook 그룹 (Wedding String Quartet)
  - 지인 공유 (오케스트라 동료)
- [ ] **첫 매출 $100+ 달성**
**완료 조건:** 판매 시스템 가동 + 첫 $100 수익

## Phase 3: 성장 및 브랜드 구축 (4-8개월)

### M6. 카탈로그 30개 돌파 [편곡]
**목표:** 안정적인 수익 기반 구축
- [ ] 추가 20개 편곡 제작
  - 베스트셀러 장르 집중 (데이터 기반)
  - K-pop 특화 또는 웨딩송 특화 선택
- [ ] 판매 데이터 분석
  - 가장 많이 팔린 곡/편성 파악
  - 가격 최적화 테스트
  - 리뷰 및 피드백 수집
- [ ] **월 순이익 $500+ 달성**
  - Passive income 비중 증가
  - 재구매 고객 확보
**완료 조건:** 30개 카탈로그 + 월 $500

### M7. 커스텀 편곡 서비스 론칭 [편곡]
**목표:** Active Income 추가 (고단가)
- [ ] 의뢰 시스템 구축
  - 웹사이트에 "커스텀 편곡" 페이지 추가
  - Google Form 또는 이메일 접수
  - 의뢰서 템플릿 작성
- [ ] 가격 정책 공개
  - 기본: $200 (3-4분 곡)
  - 복잡: $300-500 (긴 곡, 특수 요청)
  - 납기: 2주 (급행 +50%)
- [ ] 첫 유료 커스텀 의뢰 완료
  - 포트폴리오에 추가 (허락 시)
  - 리뷰 확보
**완료 조건:** 커스텀 서비스 오픈 + 첫 의뢰 완료

### M8. SNS 및 샘플 영상 제작 [마케팅]
**목표:** 브랜드 인지도 확보
- [ ] YouTube/Instagram 계정 개설
  - 채널명 선정 (예: "TK Arrangements")
  - 프로필 및 소개 최적화
- [ ] 자체 편곡 연주 영상 업로드
  - 본인 직접 연주 (바이올린 파트)
  - 또는 오케스트라 동료와 협업
  - 3-5개 영상 (BTS, 디즈니, 웨딩송)
- [ ] SEO 최적화
  - 제목: "BTS Spring Day String Quartet Sheet Music"
  - 설명란에 구매 링크
  - 태그: #stringquartet #sheetmusic #kpop
- [ ] 팔로워/구독자 100명 돌파
**완료 조건:** 3개 영상 + 100 팔로워

## Phase 4: 스케일업 및 다각화 (8-12개월)

### M9. 채보 서비스 추가 [채보 신규]
**목표:** 프리미엄 서비스 라인 확장
- [ ] AI AMT 도구 테스트
  - AnthemScore, Basic Pitch, Demucs 비교
  - 정확도 검증 (70-80% 목표)
  - 수동 보정 워크플로우 확립
- [ ] "악보 없는 곡 채보해드립니다" 서비스 론칭
  - 가격: $300+ (기술적 난이도 높음)
  - 타겟: 인디곡, 라이브 버전, 커버곡
- [ ] 첫 채보 의뢰 완료
  - 포트폴리오에 추가
  - 채보 vs 편곡 수익성 비교
**완료 조건:** 채보 워크플로우 확립 + 첫 의뢰 완료

### M10. Passive Income 안정화 및 자동화 [스케일]
**목표:** 지속 가능한 수익 구조 완성
- [ ] 카탈로그 50개 돌파
  - 다양한 장르/편성 확보
  - 롱테일 수익 발생
- [ ] **월 수익 $1,000+ 달성**
  - Passive (카탈로그 판매): $600+
  - Active (커스텀): $400+
- [ ] AI 보조 편곡 실험
  - AI 툴로 초안 생성 (시간 30-50% 단축)
  - 본인은 검수 및 음악적 다듬기
- [ ] 외주 편곡가 테스트
  - 중급 편곡가 1명 고용 (의뢰당 $50-100)
  - 대식님은 최종 검수만
  - 스케일업 가능성 검증
- [ ] 전업 전환 또는 추가 스케일업 결정
  - 수익 안정성 평가
  - 시간 투자 대비 수익 분석
**완료 조건:** 월 $1,000 + 자동화/외주 시스템 테스트 완료
