# 🎻 악보 프로젝트

**대중음악 편곡 악보 제작 및 판매 사업**

---

## 📋 프로젝트 개요

- **목표:** 부업으로 시작 → 12개월 내 월 $1,000+ 수익
- **핵심 전략:** 편곡 90% (기존 악보 → 다른 편성) + 채보 10% (음원 → 악보)
- **타겟 시장:** 대중음악 string quartet/violin duo (웨딩, K-pop, 애니OST)
- **차별화:** 프로 연주자의 "연주 가능성" + 음악적 감성

---

## 📂 프로젝트 구조

```
music-project/
├── README.md              ← 지금 보는 파일 (전체 개요)
├── MILESTONES.md          ← 10개 마일스톤 (0-12개월 로드맵)
├── ACTION-PLAN.md         ← 구체적 실행 가이드 (당장 시작 가능)
├── quality-checklist.md   ← 편곡 품질 체크리스트 (만들 예정)
├── time-log.md            ← 작업 시간 기록 (만들 예정)
└── catalog/               ← 완성된 악보 파일들 (만들 예정)
```
