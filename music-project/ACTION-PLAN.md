### Step 3: 첫 편곡 실행 (3-5시간)

**편곡 체크리스트:**

#### A. 음역대 배분
- Violin I: E4-E7 (주 멜로디, 화려한 부분)
- Violin II: G3-A6 (하모니, 카운터 멜로디)
- Viola: C3-E6 (내성, 리듬 보강)
- Cello: C2-C5 (베이스, 화성 기반)

#### B. 역할 분담
1. **인트로/엔딩:** 어느 악기가 리드?
2. **A 섹션 (verse):** 멜로디 vs 반주
3. **B 섹션 (chorus):** 풀 하모니 (4파트 다 활용)
4. **브릿지:** 텍스처 변화 (예: Violin solo + pizzicato 반주)

#### C. 연주 가능성
- [ ] 포지션 이동 무리 없는지 체크
- [ ] 빠른 패시지에서 운지 가능한지
- [ ] 보잉 방향 표기 (∏ = down-bow, V = up-bow)
- [ ] 다이내믹스 (p, mf, f) 표기

#### D. 최종 점검
- [ ] 각 파트 MIDI 재생해보기
- [ ] 전체 밸런스 확인
- [ ] PDF 출력 (Full Score + 4개 개별 파트)

**액션:**
- [ ] 3-5시간 집중 작업 (주말 활용)
- [ ] 완성 후 본인이 바이올린 파트 연주해보기
- [ ] 오케스트라 동료 1명에게 피드백 요청

## 🎯 다음: M3 품질 기준 확립

### 연주 가능성 체크리스트 작성

```markdown
## 편곡 품질 체크리스트

### 1. 음역대 (Range Check)
- [ ] Violin I: E4-E7 범위 내
- [ ] Violin II: G3-A6 범위 내
- [ ] Viola: C3-E6 범위 내
- [ ] Cello: C2-C5 범위 내
- [ ] 극단적 음역(High E 이상) 사용 시 필수성 확인

### 2. 기술적 연주 가능성
- [ ] 포지션 이동: 무리한 점프 없음
- [ ] 핑거링: 빠른 패시지에서 실현 가능
- [ ] 보잉: 긴 레가토 구간에 호흡 표시
- [ ] 더블 스탑: 실제 연주 가능한 조합만
- [ ] 템포: 빠른 곡에서 16분음표 너무 많지 않게

### 3. 음악적 완성도
- [ ] 파트 밸런스: 어느 한 파트가 너무 쉽거나 어렵지 않음
- [ ] 다이내믹스: 명확히 표기 (p, mp, mf, f, ff)
- [ ] 아티큘레이션: 레가토, 스타카토, 악센트 표기
- [ ] 프레이징: 음악적 흐름 자연스러움
- [ ] 클라이맥스: 감정선 잘 표현됨

### 4. 악보 가독성
- [ ] 보표 간격 적절
- [ ] 페이지 넘김 위치 고려 (긴 쉼표 활용)
- [ ] 리허설 마크 또는 소절 번호
- [ ] 템포, 박자, 조표 명확
- [ ] 저작권 및 편곡자 표기

### 5. 최종 검증
- [ ] MIDI 재생 청음
- [ ] 본인 연주 또는 동료 리뷰
- [ ] 오타/오류 최종 확인
- [ ] PDF 출력 테스트
```

**액션:**

## 💡 팁 & 주의사항

### 편곡할 때
- 처음엔 간단한 곡부터 (3-4분, 단순한 화성)
- 원곡의 "느낌"을 살리는 게 핵심 (음표만 옮기기 X)
- 각 파트가 "재미있게" 연주할 수 있어야 함 (Viola도!)

### 저작권
- 퍼블릭 도메인 확인: 작곡가 사망 후 70년 (미국 기준)
- 최신곡: Musicnotes 같은 곳에서 악보 구매 → 편곡은 OK (재판매 전 확인)
- 의심스러우면 변호사 상담 (나중에 규모 커지면)

### 시간 관리
- 본업(ASML) 우선, 주당 10-15시간 투자
- 주말 집중 작업 + 평일 1-2시간
- 번아웃 방지: 월 2-3곡 목표로 꾸준히
