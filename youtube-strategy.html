🎻 Tuxedo Horse - YouTube 채널 전략

채널 콘셉트: 턱시도 + 말 가면을 착용한 정체불명의 프로 바이올리니스트가 전세계 히트곡을 커버
목표: 부수입 최대화 (풀타임 직장과 병행)
핵심 강점: 프로급 연주 실력 + 독특한 비주얼 + 완벽한 익명성

🎯 곡 선정 전략 (수익 극대화)

카테고리 비중 전략 예시
K-POP 40% 신곡 48시간 내 커버 NewJeans, aespa, BTS
애니/영화 OST 30% 충성도 높은 팬덤 귀멸의 칼날, Interstellar
Bollywood 20% 인도 시장 공략 Dil Diyan Gallan
TikTok 바이럴 10% 실시간 트렌드 주간 trending 곡

📅 주간 제작 스케줄 (3-4시간/주)

일요일: 촬영의 날 (2-3시간)

  • 한번에 여러 곡 촬영 - 의상 세팅 1회
  • 롱폼 1개 (3-5분) + 쇼츠 소스 3-4개
  • 같은 조명/배경으로 연속 촬영

평일: 편집 (하루 30분)

  • 월: 롱폼 편집
  • 화/수/목: 쇼츠 3개 편집

🎬 제작 워크플로우

1. MR 준비 (5-10분)
• YouTube에서 "곡명 + instrumental" 검색
• 또는 LALAL.AI로 보컬 제거 (무료 10분/월)

2. 녹음 (30분)
• MR 들으며 바이올린 녹음
• 소프트웨어: GarageBand (Mac) / BandLab (모바일)

3. 촬영 (20분/곡)
• 3개 앵글: 미디엄 샷 / 클로즈업 / 와이드
• 녹음된 오디오에 립싱크

4. 편집 (40분)
• 영상 + 오디오 싱크
• 썸네일 제작
• CapCut (모바일) / DaVinci Resolve (무료, PC)

💰 수익 예상 (6개월 로드맵)

시기 구독자 월 수익 수익원
Month 1-2 0 → 5,000 $0 성장기
Month 3-4 5,000 → 15,000 $200-400 AdSense + 악보판매
Month 5-6 15,000 → 50,000 $800-1,500 AdSense + 악보 + 스트리밍

⚖️ 저작권 전략

✅ 현실: 커버 곡은 합법
• YouTube가 자동 인식 → "저작권 클레임" 발생
• 영상은 유지, 수익만 원작자와 50-70% 공유
• 영상 삭제는 거의 없음

📌 대응:
• 초기엔 클레임 받아들이고 구독자 확보 우선
• 구독자 늘면 스폰서십으로 수익 보충
• K-POP/애니 OST는 클레임만, 차단 거의 없음

🚀 첫 3개월 실행 플랜

Week 1-2: 런칭

  • 채널 개설 (Tuxedo Horse)
  • K-POP 신곡 커버 (NewJeans 등)
  • 쇼츠 4개 + 롱폼 1개

Week 3-4: 다양성

  • Bollywood 명곡 (Dil Diyan Gallan)
  • 애니 OST (귀멸의 칼날)

Month 2-3: 일관성 유지

  • 주 2-3개 쇼츠 꾸준히
  • 트렌드 곡 빠르게 반응
  • 크로스 포스팅 (TikTok, Instagram)

📝 성공 포인트

⚡ 일관성 > 완벽함 - 매주 정해진 날에 업로드
